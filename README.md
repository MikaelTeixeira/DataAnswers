# Mensagens Automaticas

Aplicacao web para criar, organizar e usar respostas prontas por plataforma. O sistema permite cadastrar usuarios, separar perfis de administrador e usuario comum, gerenciar plataformas e manter respostas dinamicas salvas em banco de dados MySQL.

As mensagens podem usar campos automaticos como `@responsible_name` e `@student_name`, preenchidos conforme os dados informados no dashboard. Administradores tambem podem visualizar usuarios cadastrados e controlar quais contas possuem acesso administrativo.
